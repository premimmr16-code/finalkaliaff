📘 Admin Guide

🍛 Menu
Menu Manage থেকে খাবার add/edit করুন। Category select করুন, price/discount/photo দিন।
📦 Orders
Order detail দেখুন, status update করুন, customer-কে WhatsApp/Gmail manual message পাঠান।
🖼️ Banner / Notice
Homepage banner ও notice board update করে offer/announcement দেখান।
🎟️ Coupon
Percentage, fixed discount বা free delivery coupon create করুন।
🏆 Loyalty
Point target, max point, referral/review bonus ও free delivery minimum order control করুন। User side শুধু simple point দেখবে।
👥 Users
Customer list, total order, spend, point ও reward status monitor করুন।
👨‍💼 Support Admin
Support team add/remove করুন। Chat/order support permission note রাখুন।
🔔 Notification
Settings থেকে admin WhatsApp/Gmail দিন। Browser notification allow করলে new order alert আসবে।

🔥 FINAL FIREBASE RULES — একবার সেট করুন

Firebase Console → Firestore Database → Rules তে নিচের rules দিন:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

Firebase Console → Storage → Rules তে নিচের rules দিন:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
